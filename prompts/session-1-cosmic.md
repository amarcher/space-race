# Session 1 · Cosmic · 15 cards

**Chat-based tool (Gemini app, etc.):** do this whole session in ONE chat. The anchor (Prompt 01) locks the style; every subsequent prompt references it via conversation continuity.

**Shot-based tool (Google Flow, Imagen Studio):** no chat needed. Every prompt attaches `s1-01-black-hole_v1.png` as a reference image / Ingredient. Order doesn't matter, but subject sequence is still useful because you re-anchor the palette mentally as you go.

**Workflow per prompt:**
1. Paste the prompt
2. Generate 3–4 variants
3. Pick the winner
4. Save to `artbin/` with the suggested filename
5. Move to the next prompt

**Aspect ratio: 3:4 portrait for ALL 15 cards.** Art is now **full-bleed** — extends edge-to-edge on the printed card, with text overlays on top. Set in UI if available (3:4 is in Gemini's dropdown); otherwise add *"3:4 portrait aspect ratio, subject centered, cosmic field filling the full frame edge-to-edge"* to each prompt.

**Every 3–4 prompts, re-anchor the palette** with: *"Keep the palette strictly to Void #06061A, Nebula #1A0933, plasma magenta #C879FF, ember red #FF5E5B. No other colors."*

## Prompt 01 · Black Hole · ⭐ THE ANCHOR

Save as: `s1-01-black-hole_v1.png`

```
Painterly cinematic illustration of a dramatic black hole with gravitational lensing warping surrounding starlight into a clean Einstein ring. Deep cosmic space setting, moody volumetric lighting with plasma-magenta #C879FF highlights along the accretion disk. Retro-futurist space-opera aesthetic inspired by Chesley Bonestell and Denis Villeneuve's Dune. Palette dominated by Void #06061A and Nebula #1A0933. Dramatic chiaroscuro, shallow depth of field, subtle film grain. 3:4 portrait aspect ratio, subject centered, cosmic field filling the full frame edge-to-edge for full-bleed card printing. No text, no logos, no UI. 300dpi, print-ready.
```

**Pick the variant with:**
- Clearest gravitational lensing / Einstein ring
- Deepest true-black Void (not washed violet)
- Dramatic plasma-magenta highlights (not pink, not purple)
- No AI artifacts (distorted geometry, duplicated stars)
- Subject sits cleanly in center third

**Then:** upload the winning variant back into this same chat as an attachment. That becomes the reference for prompts 02–15.
